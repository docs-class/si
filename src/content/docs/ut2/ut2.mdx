### ** Posibles contenidos basados en los criterios de evaluación**

#### **a) Se han identificado los elementos funcionales de un sistema informático.**
**Contenidos:**
- **Componentes de un sistema informático**: Hardware (CPU, RAM, almacenamiento, dispositivos de entrada y salida) y Software (sistemas operativos y aplicaciones).
- **Relación entre hardware y software**: Cómo interactúan ambos elementos para ejecutar procesos y tareas.
- **Tipos de hardware y sus funciones**: Dispositivos de entrada (teclado, ratón), dispositivos de salida (pantalla, impresora) y dispositivos de almacenamiento (discos duros, SSD, unidades ópticas).

#### **b) Se han analizado las características, funciones y arquitectura de un sistema operativo.**
**Contenidos:**
- **Funciones de un sistema operativo**: Gestión de procesos, gestión de memoria, gestión de archivos, gestión de dispositivos y control de seguridad.
- **Arquitectura de los sistemas operativos**: Núcleo (kernel), espacio de usuario, API, controladores de dispositivos.
- **Tipos de sistemas operativos**: Sistemas operativos monolíticos, basados en microkernel, sistemas distribuidos.
- **Características de los principales sistemas operativos**: Windows, Linux, macOS, sistemas embebidos, y sus diferencias.

#### **c) Se han comparado sistemas operativos en base a sus requisitos, características, campos de aplicación y licencias de uso.**
**Contenidos:**
- **Requisitos de hardware y software de diferentes sistemas operativos**: Comparar recursos mínimos y recomendados (memoria RAM, CPU, almacenamiento).
- **Comparación de características**: Multitarea, multiusuario, administración de seguridad, interfaz gráfica de usuario (GUI) vs. interfaz de línea de comandos (CLI).
- **Licencias de uso**: Software propietario (Windows, macOS) vs. Software libre (Linux, FreeBSD) y sus implicaciones.
- **Aplicaciones y campos de uso**: Sistemas operativos para servidores, estaciones de trabajo, dispositivos móviles y sistemas embebidos.

#### **d) Se ha planificado el proceso de la instalación de sistemas operativos.**
**Contenidos:**
- **Planificación de una instalación**: Requisitos previos, particionado del disco, opciones de instalación (limpia o actualización), elección de sistema de archivos (FAT32, NTFS, ext4).
- **Creación de medios de instalación**: USB booteable, DVD o instalación en red.
- **Consideraciones de seguridad y rendimiento**: Asignación de recursos, actualización de BIOS/UEFI y configuración de hardware previo a la instalación.

#### **e) Se han instalado y actualizado sistemas operativos libres y propietarios.**
**Contenidos:**
- **Instalación de sistemas operativos propietarios y libres**: Instalación de Windows y distribución de Linux (Ubuntu, Fedora).
- **Proceso de instalación paso a paso**: Creación de particiones, elección de sistema de archivos, instalación de controladores y configuración inicial.
- **Actualización de sistemas operativos**: Tipos de actualizaciones (mayores, menores, parches de seguridad) y métodos de actualización en Windows y Linux (Windows Update, apt-get, yum).

#### **f) Se han aplicado técnicas de actualización y recuperación del sistema.**
**Contenidos:**
- **Métodos de actualización de software**: Actualización manual y automática de controladores, parches de seguridad y actualizaciones de sistema.
- **Recuperación del sistema**: Creación y restauración de puntos de restauración en Windows, uso de discos de recuperación, reinstalación sin pérdida de datos.
- **Recuperación ante fallos**: Uso de herramientas de reparación del sistema (chkdsk, sfc), restauración de arranque y corrección de errores en el sistema de archivos.

#### **g) Se han utilizado tecnologías de virtualización para instalar y probar sistemas operativos.**
**Contenidos:**
- **Conceptos de virtualización**: Máquina virtual, hipervisor (tipo 1 y tipo 2).
- **Software de virtualización**: VirtualBox, VMware, Hyper-V.
- **Creación de máquinas virtuales**: Asignación de recursos (CPU, memoria RAM, disco duro virtual), configuración de red y dispositivos virtuales.
- **Pruebas de sistemas operativos en entornos virtuales**: Instalación y configuración de diferentes sistemas operativos en máquinas virtuales para pruebas y formación.

#### **h) Se han instalado, desinstalado y actualizado aplicaciones.**
**Contenidos:**
- **Instalación de aplicaciones**: Métodos de instalación en diferentes sistemas operativos (paquetes .exe, .msi, .deb, .rpm).
- **Desinstalación de aplicaciones**: Uso de herramientas de desinstalación en Windows y Linux (Programas y Características, apt-get remove, yum remove).
- **Actualización de aplicaciones**: Configuración de actualizaciones automáticas, descarga manual de nuevas versiones y actualización mediante gestores de paquetes.

#### **i) Se han documentado los procesos realizados.**
**Contenidos:**
- **Documentación técnica**: Redacción de procedimientos detallados para la instalación y configuración de sistemas operativos y aplicaciones.
- **Creación de informes y guías**: Documentar los pasos de configuración, incidencias y resolución de problemas.
- **Herramientas de documentación**: Uso de herramientas como Markdown, Microsoft Word, Google Docs o wikis para organizar la información.
